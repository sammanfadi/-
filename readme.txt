== الوصف ==

قالب محتوى هو قالب ووردبريس احترافي مصمم خصيصاً لعرض المتاجر والخدمات المحلية على خريطة تفاعلية. 
يوفر القالب تجربة مستخدم متميزة مع إمكانيات بحث وفلترة متقدمة، ونظام تقييمات شامل، ولوحة تحكم مخصصة.

== المميزات الرئيسية ==

* خريطة تفاعلية متقدمة باستخدام Leaflet
* نظام إدارة المتاجر الشامل
* تصميم متجاوب ومتوافق مع جميع الأجهزة
* نظام تقييمات ومراجعات
* بحث وفلترة متقدمة
* تحسين محركات البحث (SEO) والسيو المحلي
* دعم كامل للغة العربية (RTL)
* لوحة تحكم مخصصة
* إعدادات تخصيص شاملة
* ودجات مخصصة
* اختصارات مفيدة للمحتوى
* Schema Markup للمتاجر المحلية
* تحسين الأداء وسرعة التحميل

== متطلبات النظام ==

* ووردبريس 5.0 أو أحدث
* PHP 7.4 أو أحدث
* MySQL 5.6 أو أحدث

== التثبيت ==

1. قم بتحميل ملفات القالب إلى مجلد /wp-content/themes/
2. فعل القالب من لوحة تحكم ووردبريس
3. اذهب إلى "قالب محتوى" > "لوحة التحكم" لبدء الإعداد
4. أضف فئات المتاجر من "إدارة الفئات"
5. أضف المواقع الجغرافية من "إدارة المواقع"
6. ابدأ بإضافة المتاجر الأولى

== الإعداد السريع ==

1. **إعداد الخريطة:**
   - اذهب إلى "قالب محتوى" > "إعدادات الخريطة"
   - حدد الموقع الافتراضي للخريطة
   - اختر مستوى التكبير المناسب

2. **تخصيص المظهر:**
   - اذهب إلى "المظهر" > "تخصيص"
   - أضف شعار الموقع
   - اختر الألوان والخطوط
   - أضف معلومات الاتصال

3. **إضافة المحتوى:**
   - أضف فئات المتاجر مع الأيقونات والألوان
   - أضف المناطق الجغرافية
   - ابدأ بإضافة المتاجر مع تحديد مواقعها

== الاختصارات المتاحة ==

* `[mohtawa_map]` - عرض خريطة المتاجر
* `[mohtawa_stores]` - عرض قائمة المتاجر
* `[mohtawa_categories]` - عرض فئات المتاجر
* `[mohtawa_search_form]` - نموذج البحث
* `[mohtawa_contact_info]` - معلومات الاتصال

== الودجات المتاحة ==

* المتاجر المميزة

== أنواع المحتوى المخصصة ==

* **المتاجر (Stores):**
  - العنوان والوصف
  - معلومات الاتصال
  - الموقع على الخريطة
  - ساعات العمل
  - الخدمات المتاحة
  - معرض الصور
  - التقييمات

* **التصنيفات:**
  - فئات المتاجر مع الأيقونات والألوان
  - المواقع الجغرافية

== الحقول المخصصة ==

* store_address - عنوان المتجر
* store_phone - رقم الهاتف
* store_email - البريد الإلكتروني
* store_website - الموقع الإلكتروني
* store_latitude - خط العرض
* store_longitude - خط الطول
* store_rating - التقييم
* store_hours - ساعات العمل
* store_services - الخدمات المتاحة
* store_featured - متجر مميز
* store_is_open - حالة المتجر
